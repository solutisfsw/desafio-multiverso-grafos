DESAFIO RICK - GRAFOS

Documentação

Descrição geral: este algoritmo varre o nó inicial, verificando se já chegou no destino ou, caso contrário, se o nó seguinte tem mais links, armazenando-os em um vetor para ser visitado posteriormente. O código faz isso até não restar nós a serem visitados, gerando todos os caminhos possíveis até o nó alvo. A respostas às questões dos desafios são realizadas posteriormente a geração das rotas.

Como Executar

  • Abra o arquivo index.html, no navegador;
  • As respostas irão aparecer na tela inicial.

Classes

Graph.js
Essa classe representa o Grafo. Nodes é um conjunto de nós do tipo Node. 
	function Graph(nodes){
		this.nodes = nodes;
	}
Métodos:
getNodes: Retorna todos os nós do grafo
findAllWays: Retorna um caminho entre um nó e outro nó
getRouteToYPassingX:Retorna rotas de um nó a outro passando por um nó x (Way[], Node, Node)
getCountRouteToYMaxStop: Retorna quantidade de rotas de um nó a outro com até x paradas (Way[], int, Node)
getCountRouteToYMaxDistance: Retorna quantidade de rotas de um nó a outro com uma distânica até x (Way[], int, Node)
getLessRoute: Obtém as menores rotas de um nó a outro (Way[])
printRoutes: Imprime rotas
getAllRoutes: Obtém todas as rotas de um nó a outro (Node, Node)

Node.js
Essa classe representa um nó do grafo. Links é um conjunto de links entre os nós, na direção deste nó para o próximo nó. Label é a label do nó, ex: A, B, E. 
	function Node(label){
		this.links = [];
		this.label = label;
	}
Métodos:
getLinks: obtem a lista de links (Array)
addLink: adiciona um link a um nó (Void)
getLabel: retorna o label do nó (String)

Link.js
Essa classe representa uma aresta nó do grafo. End é o nó que a aresta liga e weight o peso da aresta. 
	function Link (end, weight){
		this.end = end;
		this.weight = weight;
	}
